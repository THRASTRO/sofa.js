# eraPdp

```js
rv = ERFA.pdp(a, b)
```

p-vector inner (=scalar=dot) product.

## Given:
```
   a      double[3]     first p-vector
   b      double[3]     second p-vector
```

## Returned (function value):
```
          double        a . b
```
