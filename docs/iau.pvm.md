# iauPvm

```js
[r, s] = IAU.pvm(pv)
```

Modulus of pv-vector.

## Given:
```
   pv     double[2][3]   pv-vector
```

## Returned:
```
   r      double         modulus of position component
   s      double         modulus of velocity component
```

## Called:
```
   iauPm        modulus of p-vector
```
