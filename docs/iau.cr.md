# iauCr

```js
c = IAU.cr(r)
```

Copy an r-matrix.

## Given:
```
   r        double[3][3]    r-matrix to be copied
```

## Returned:
```
   c        double[3][3]    copy
```

## Called:
```
   iauCp        copy p-vector
```
