# iauCpv

```js
c = IAU.cpv(pv)
```

Copy a position/velocity vector.

## Given:
```
   pv     double[2][3]    position/velocity vector to be copied
```

## Returned:
```
   c      double[2][3]    copy
```

## Called:
```
   iauCp        copy p-vector
```
