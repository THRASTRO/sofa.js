# eraS2c

```js
c = ERFA.s2c(theta, phi)
```

Convert spherical coordinates to Cartesian.

## Given:
```
   theta    double       longitude angle (radians)
   phi      double       latitude angle (radians)
```

## Returned:
```
   c        double[3]    direction cosines
```
