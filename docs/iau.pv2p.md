# iauPv2p

```js
p = IAU.pv2p(pv)
```

Discard velocity component of a pv-vector.

## Given:
```
   pv      double[2][3]     pv-vector
```

## Returned:
```
   p       double[3]        p-vector
```

## Called:
```
   iauCp        copy p-vector
```
