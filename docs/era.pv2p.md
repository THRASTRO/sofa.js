# eraPv2p

```js
p = ERFA.pv2p(pv)
```

Discard velocity component of a pv-vector.

## Given:
```
   pv      double[2][3]     pv-vector
```

## Returned:
```
   p       double[3]        p-vector
```

## Called:
```
   eraCp        copy p-vector
```
