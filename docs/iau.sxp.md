# iauSxp

```js
sp = IAU.sxp(s, p)
```

Multiply a p-vector by a scalar.

## Given:
```
   s      double        scalar
   p      double[3]     p-vector
```

## Returned:
```
   sp     double[3]     s * p
```

## Note:
```
   It is permissible for p and sp to be the same array.
```
