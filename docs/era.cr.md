# eraCr

```js
c = ERFA.cr(r)
```

Copy an r-matrix.

## Given:
```
   r        double[3][3]    r-matrix to be copied
```

## Returned:
```
   c        double[3][3]    copy
```

## Called:
```
   eraCp        copy p-vector
```
